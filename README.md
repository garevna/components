# dgtek-styles

##### Styles should be utilized in dgtek.net projects only.

### install component
```
yarn add dgtek-styles
```
### Import styles

###### main.js
```
import 'dgtek-styles/css/fonts.scss'
import 'dgtek-styles/css/variables.scss'
```

###### App.vue
```
import 'dgtek-styles/css/main.css'
```
or simply
```
import 'dgtek-styles'
```
